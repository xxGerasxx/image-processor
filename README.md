# Image Processor Application

Десктопное приложение для обработки изображений на Java с использованием OpenCV и Webcam Capture.

## Функции

- Загрузка изображений (PNG, JPG)
- Снимок с веб-камеры
- Отображение цветовых каналов (RGB)
- Негатив
- Размытие по Гауссу
- Рисование круга с заданными координатами

## Сборка и запуск

### Требования
- JDK 11 или выше
- Maven 3.6 или выше

### Сборка
```bash
mvn clean compile
mvn package assembly:single
